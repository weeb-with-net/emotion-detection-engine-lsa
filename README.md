# Emotion Detection & Learning Support Engine

An AI app that reads a student's description of a study problem, detects the
emotion behind it (Bored, Confident, Confused, Curious, Frustrated), and uses
Gemini AI to generate a supportive, personalized response.

## How this repo is organized

This repo has two kinds of content living side by side:

- **The 8 numbered folders** (`1. Brainstorming & Ideation` ... `8. Project
  Demonstration`) — these are **college submission paperwork**, filled in near
  the end of the project. They are not code.
- **Everything else at the root** (`app.py`, `src/`, `models/`, `data/`,
  `requirements.txt`, etc.) — this is the **actual working application**,
  built and committed continuously as we go.

## Setup

See `verify_setup.py` and the Epic 1 setup steps for getting your environment
running (Python, dependencies, `.env` with your Gemini API key).
